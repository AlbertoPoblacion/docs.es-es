# <a name="lambda-expression-cannot-be-converted-to-typename-because-type-typename-is-declared-mustinherit-and-cannot-be-created"></a>Expresión lambda no se puede convertir a '\<typename >' porque tipo '\<typename >' está declarado como 'MustInherit' y no se puede crear
Una instrucción intenta convertir una expresión lambda a un tipo que solo puede ser una clase base y no se puede usar para crear una instancia.  
  
 **Identificador de error:** BC36626
