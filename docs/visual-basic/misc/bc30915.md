# <a name="name-cannot-expose-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-outside-the-project-through-specifier-type"></a>'\<nombre >' no puede exponer el tipo delegado subyacente '\<delegatetype >' del evento que se implementa fuera del proyecto a través de \<especificador > '\<tipo >'
El código expone un tipo privado fuera de la clase pública.  
  
 **Identificador de error:** BC30915  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Declare el tipo `Public`.  
  
     -o bien-  
  
- Utilice otro tipo.
