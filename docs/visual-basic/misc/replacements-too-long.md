# <a name="replacements-too-long"></a>Los reemplazos son demasiado largos
Ha especificado un reemplazo que supera la longitud permitida.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Consulte la documentación del componente para obtener información sobre la restricción de longitud.
