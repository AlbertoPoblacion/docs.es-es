# <a name="expected-closing--for-xml-cdata-section"></a>Se esperaba el cierre ']] >' para la sección CDATA XML
A un literal de XML CDATA le faltan los caracteres de cierre ']]>' necesarios.  
  
 **Identificador de error:** BC31162  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Agregue los caracteres de cierre ']]>' necesarios al final del literal de XML CDATA.
