# <a name="web-methods-cannot-be-evaluated-in-debug-windows"></a>Los métodos web no se pueden evaluar en ventanas de depuración
Los métodos de servicio web XML no se pueden probar dentro de la ventana de depuración.  
  
 **Identificador de error:** BC30732  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Pruebe los métodos de servicio web XML desde el Editor de código.
