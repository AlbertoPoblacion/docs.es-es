# <a name="white-space-may-not-appear-here"></a>Aquí no puede aparecer un espacio en blanco.
Uno o varios espacios se incluyen en el nombre de un literal de elemento XML.  
  
 **Identificador de error:** BC31177  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite el espacio o los espacios del nombre del elemento XML literal.
