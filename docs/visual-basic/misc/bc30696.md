# <a name="membername1-and-membername2-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>'\<nombredemiembro1 >' y '\<nombredemiembro2 >' no se pueden sobrecargar entre sí porque se diferencian únicamente por los tipos de parámetros opcionales
Los miembros sobrecargados deben tener tipos de datos diferentes para uno o varios parámetros requeridos.  
  
 **Identificador de error:** BC30696  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Defina al menos un parámetro requerido como un tipo de datos diferente.
