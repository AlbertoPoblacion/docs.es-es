# <a name="optional-parameters-cannot-have-structure-types"></a>Los parámetros opcionales no pueden tener tipos de estructura.
Los parámetros opcionales no pueden tener tipos de estructura.  
  
 **Identificador de error:** BC31405  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Declare los parámetros de tipo de estructura según sea necesario.  
  
2. Declare un miembro de clase estática para ocupar el lugar de una estructura.
