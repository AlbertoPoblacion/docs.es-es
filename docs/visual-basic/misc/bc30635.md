# <a name="readonly-and-writeonly-cannot-be-combined"></a>'ReadOnly' y 'WriteOnly' no se pueden combinar
Los modificadores `ReadOnly` y `WriteOnly` se excluyen mutuamente.  
  
 **Identificador de error:** BC30635  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Proporcione `ReadOnly` o `WriteOnly`, pero no ambos.
