# <a name="unable-to-create-xml-documentation-file-name-message"></a>No se puede crear el archivo de documentación XML '\<nombre >': \<mensaje >
No se puede crear el archivo de documentación XML debido a un error.  
  
 **Identificador de error:** BC42311  
  
 Los mensajes siguientes se pueden mostrar con este error:  
  
- [Bloque de comentario XML no puede asociarse a cualquier elemento del lenguaje que admita la aplicación de comentarios de documentación XML](../../visual-basic/misc/bc42312.md).  
  
- [No se permite la etiqueta de comentario XML 'returns' en una propiedad 'WriteOnly'](../../visual-basic/misc/bc42313.md).  
  
- [El comentario XML no se puede aplicar más de una vez en una parcial \<tipo >](../../visual-basic/misc/bc42314.md).  
  
- [La etiqueta de comentario XML 'returns' no se permite en un elemento de lenguaje 'declare sub'](../../visual-basic/misc/bc42315.md).  
  
- [Error de análisis de documentación XML: Etiqueta de apertura '\<etiqueta >' no tiene la correspondiente etiqueta de cierre](../../visual-basic/misc/bc42316.md).  
  
- [El parámetro de tipo de comentario XML \<parámetro > no coincide con un parámetro de tipo en la correspondiente \<palabra clave > instrucción](../../visual-basic/misc/bc42317.md).  
  
- [El parámetro de tipo de comentario XML debe tener un atributo 'name'](../../visual-basic/misc/bc42318.md).  
  
- [El comentario XML excepción debe tener un atributo 'cref'](../../visual-basic/language-reference/error-messages/xml-comment-exception-must-have-a-cref-attribute.md).  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Consulte el mensaje proporcionado para obtener recomendaciones.
