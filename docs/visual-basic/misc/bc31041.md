# <a name="statement-is-not-valid-in-an-interface"></a>Instrucción no válida en una interfaz
Se encontró una instrucción no válida en la interfaz.  
  
 **Identificador de error:** BC31041  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe que las instrucciones de la interfaz definen elementos, pero no proporcionan implementación.
