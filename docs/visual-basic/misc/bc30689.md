# <a name="statement-cannot-appear-outside-of-a-method-bodymultiline-lambda"></a>La instrucción no puede aparecer fuera de un cuerpo de método o una expresión lambda de varias líneas
Las instrucciones ejecutables deben usarse en los procedimientos.  
  
 **Identificador de error:** BC30689  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Mueva la instrucción a un procedimiento o subrutina.
