# <a name="modifier-is-not-valid-on-a-method-declaration"></a>'\<modificador >' no es válido en una declaración de método
Una instrucción `Function` o `Sub` contiene una palabra clave no válida.  
  
 **Identificador de error:** BC30242  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite la palabra clave no válida de la instrucción `Function` o `Sub` .
