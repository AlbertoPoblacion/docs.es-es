# <a name="next-expected"></a>Se esperaba 'Next'.
Una construcción `On Error Resume` se realiza sin necesidad de una conclusión de `Next`.  
  
 **Identificador de error:** BC30003  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Agregue la palabra clave `Next` a la instrucción `On Error` .
