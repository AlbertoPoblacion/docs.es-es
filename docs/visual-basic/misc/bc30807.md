# <a name="let-and-set-assignment-statements-are-no-longer-supported"></a>Ya no se admiten las instrucciones de asignación 'Let' y 'Set'.
No se puede usar`Let` y `Set` en asignaciones.  
  
 **Identificador de error:** BC30807  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Omita `Let` y `Set` en las instrucciones de asignación; por ejemplo:  
  
     `X = 2`
