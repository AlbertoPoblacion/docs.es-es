# <a name="property-propertyname-cannot-be-set-to-nothing"></a>Propiedad \<propertyname > no se puede establecer en Nothing
La propiedad requiere un valor distinto de `Nothing`. Esto puede ser el resultado del cálculo incorrecto del valor.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Proporcione un valor válido.
