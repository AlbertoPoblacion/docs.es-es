# <a name="implementsclause-cannot-implement-typename-because-typename-is-a-reserved-name"></a>'\<implementsclause >' no puede implementar '\<typename >' porque '\<typename >' es un nombre reservado
Una instrucción `Implements` especificó un nombre reservado.  
  
 **Identificador de error:** BC31415  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie el nombre del miembro de interfaz que no se puede implementar.
