# <a name="no-default-member-found-for-type-typename"></a>Se encontró ningún miembro predeterminado para el tipo '\<typename >'
Se intentó realizar una llamada `Get` o `Set` del índice enlazado en tiempo de ejecución.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Asegúrese de que el objeto tiene una propiedad predeterminada con `Get` o `Set`, según la operación que esté intentando realizar.
