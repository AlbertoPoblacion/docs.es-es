# <a name="exit-statements-are-not-valid-in-the-immediate-window"></a>Las instrucciones 'Exit' no son válidas en la ventana Inmediato
Las instrucciones`Exit` solo pueden usarse en el código fuente.  
  
 **Id. de error:** BC30713  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite las instrucciones `Exit` del código de depuración.
