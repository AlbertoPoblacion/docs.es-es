# <a name="referenced-array-element-has-a-value-of-nothing"></a>El elemento de matriz al que se hace referencia tiene un valor 'Nothing'
El elemento de matriz al que se hace referencia aún no tiene un valor especificado o se estableció previamente en `Nothing`.  
  
 **Identificador de error:** BC30744  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Compruebe que los valores de la matriz se establecen correctamente.
