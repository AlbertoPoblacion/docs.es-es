# <a name="specifier-is-not-valid-on-an-enum-declaration"></a>'\<especificador >' no es válido en una declaración Enum
Se ha usado un especificador que no es válido en una declaración `Enum` .  
  
 **Identificador de error:** BC30396  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Quite el especificador.
