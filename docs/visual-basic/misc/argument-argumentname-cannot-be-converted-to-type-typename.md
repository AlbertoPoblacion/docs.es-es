# <a name="argument-argumentname-cannot-be-converted-to-type-typename"></a>Argumento '\<argumentname >' no se puede convertir al tipo '\<typename >'
Una conversión intenta cambiar el argumento a un tipo que no es válido.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Cambie el tipo a uno que sea válido.
