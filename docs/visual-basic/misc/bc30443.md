# <a name="get-is-already-declared"></a>'Get' está ya declarado
Una instrucción `Get` aparece más de una vez en un bloque de código. La instrucción `Get` declara un procedimiento de propiedad `Get` que se utiliza para recuperar un valor de una propiedad.  
  
 **Identificador de error:** BC30443  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite la instrucción `Get` redundante.
