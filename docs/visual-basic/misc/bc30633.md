# <a name="set-statement-must-end-with-a-matching-end-set"></a>La instrucción 'Set' debe terminar con la instrucción 'End Set' correspondiente
Los procedimientos de propiedad`Set` deben terminar con una instrucción `End Set` .  
  
 **Identificador de error:** BC30633  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Asegúrese de que el procedimiento de propiedad `Set` termine con una construcción `End Set` .
