# <a name="member-is-already-declared-in-this-structure"></a>'\<miembro >' ya está declarado en esta estructura
Un miembro de la enumeración se declaró más de una vez en esta estructura.  
  
 **Identificador de error:** BC31421  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Elimine la declaración superflua.
