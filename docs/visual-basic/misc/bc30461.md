# <a name="classes-cannot-be-declared-specifier"></a>No se puede declarar clases '\<especificador >'
Ha intentado declarar una `Class` o un `Module` con un especificador no válido.  
  
 **Identificador de error:** BC30461  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el especificador.
