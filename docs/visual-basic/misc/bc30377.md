# <a name="method-methodname-is-already-declared-in-interface-interfacename"></a>Método '\<methodname >' ya está declarado en la interfaz '\<interfacename >'
Se ha declarado un método de interfaz otra vez, duplicándolo.  
  
 **Identificador de error:** BC30377  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite la declaración duplicada.
