# <a name="end-tag-tagname-expected"></a>Etiqueta de cierre \</tagName > se esperaba
La etiqueta de cierre en un literal de elemento XML no coincide con la etiqueta de apertura correspondiente.  
  
 **Identificador de error:** BC31150  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Modifique la etiqueta XML final para que coincida con la etiqueta XML de inicio del literal del elemento XML.
