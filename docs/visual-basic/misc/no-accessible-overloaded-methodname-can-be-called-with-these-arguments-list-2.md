# <a name="no-accessible-overloaded-methodname-can-be-called-with-these-arguments-without-a-narrowing-conversion-list"></a>Ninguna sobrecarga accesible '\<methodname >' se puede llamar con estos argumentos sin una conversión de restricción: \<lista >
Se ha llamado a un método sobrecargado, pero el método no coincide con la lista de argumentos proporcionada sin una conversión de restricción.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Especifique `Option Strict Off`.
  
2.  Cambie los argumentos para que coincida con una firma del método sobrecargado.
