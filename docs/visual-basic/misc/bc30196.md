# <a name="comma-expected"></a>Se esperaba una coma
Falta una coma (,) entre dos de los argumentos de una lista de argumentos.  
  
 **Identificador de error:** BC30196  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Asegúrese de que todos los argumentos están separados por comas.
