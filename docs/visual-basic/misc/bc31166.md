# <a name="expected-closing--for-xml-tag"></a>Se esperaba el cierre ' >' para la etiqueta XML
A la etiqueta XML de un literal XML le falta el carácter de cierre '>' necesario.  
  
 **Identificador de error:** BC31166  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Agregue el carácter de cierre '>' al final de la etiqueta XML para el literal XML.
