# <a name="argument-cannot-be-less-than-zero"></a>El argumento no puede ser inferior a cero
Se ha facilitado un valor inferior a cero, pero el argumento no puede ser menor que cero.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Compruebe cómo se calcula el valor en cuestión, para asegurarse de que sea correcto.
