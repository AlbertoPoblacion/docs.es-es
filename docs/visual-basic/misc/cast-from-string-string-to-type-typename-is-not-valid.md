# <a name="cast-from-string-quotstringquot-to-type-typename-is-not-valid"></a>Conversión de la cadena &quot; \<cadena >&quot; al tipo '\<typename >' no es válido
Un intento de convertir una variable de cadena a otro tipo no es válido.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Compruebe los tipos implicados en la conversión y asegúrese de que está intentando la conversión a un tipo significativo.
