# <a name="length-of-argument-argumentname-must-be-greater-than-zero"></a>Longitud del argumento '\<argumentname >' debe ser mayor que cero
Un argumento tiene un valor que es menor o igual a cero.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Proporcione un argumento que sea mayor que cero.
