# <a name="currency-is-no-longer-a-supported-type-use-the-decimal-type-instead"></a>Ya no se admite el tipo 'Currency'; use el tipo 'Decimal' en su lugar
El tipo de datos `Currency` se reemplazó por el tipo de datos `Decimal` .  
  
 **Identificador de error:** BC30803  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Utilice el tipo de datos `Decimal` en lugar de `Currency`.
