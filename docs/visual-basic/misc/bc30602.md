# <a name="interface-members-must-be-methods-properties-events-or-type-definitions"></a>Los miembros de interfaz deben ser métodos, propiedades, eventos o definiciones de tipo
Solo los eventos, las propiedades, los métodos y los tipos anidados pueden ser miembros de una interfaz.  
  
 **Identificador de error:** BC30602  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Asegúrese de que el elemento es un método, una propiedad, un evento o un tipo anidado como, por ejemplo, una enumeración.
