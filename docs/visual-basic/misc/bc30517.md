# <a name="overload-resolution-failed-because-no-method-is-accessible"></a>Error en la resolución de sobrecarga no '\<método >' es accesible
Ha realizado una llamada a un método sobrecargado, pero el compilador no encuentra un método accesible con una lista de parámetros que coincida con el número de parámetros.  
  
 **Identificador de error:** BC30517  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Compruebe la lista de argumentos que ha facilitado.  
  
2.  Compruebe el método al que está intentando llamar para determinar si es accesible.
