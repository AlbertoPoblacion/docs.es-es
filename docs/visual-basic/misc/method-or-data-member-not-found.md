# <a name="method-or-data-member-not-found"></a>No se encuentra el método o miembro de datos.
No se encontró el método o miembro de datos al que intentó acceder.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Asegúrese de que no escribió mal el nombre del miembro de datos o del método.
