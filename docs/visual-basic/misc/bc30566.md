# <a name="array-initializer-has-too-many-dimensions"></a>El inicializador de matriz tiene demasiadas dimensiones
No proporcionó suficientes dimensiones en el inicializador de la matriz.  
  
 **Identificador de error:** BC30566  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Compruebe el inicializador de matriz para determinar cuántas dimensiones son necesarias.  
  
- Quite el argumento o los argumentos adicionales.
