# <a name="argument-access-is-not-valid-valid-values-for-output-mode-are-openaccesswrite-and-openaccessdefault"></a>El argumento 'Access' no es válido. Los valores válidos para el modo Output son 'OpenAccess.Write' y 'OpenAccess.Default'
Un valor de acceso no es válido en el modo Output.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Cambie el valor a `OpenAccess.Write` u `OpenAccess.Default`.
