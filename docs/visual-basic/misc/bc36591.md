# <a name="too-many-type-arguments-to-extension-method-methodname-defined-in-typename"></a>Demasiados argumentos de tipo para el método de extensión '\<methodName >' definido en '\<typeName >'
Se invocó un método de extensión genérico con más argumentos de tipo que los parámetros de tipo que hay.  
  
 Al invocar un método genérico, debe proporcionar un argumento de tipo para cada parámetro de tipo definido por el método.  
  
 **Identificador de error:** BC36591  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite argumentos de tipo de la lista de argumentos de tipo para que haya uno para cada parámetro de tipo definido por el método genérico que está invocando.
