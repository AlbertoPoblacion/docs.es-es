# <a name="attribute-attributename-is-not-valid-error"></a>Atributo '\<attributename >' no es válido: \<error >
Common Language Runtime no acepta el atributo en el contexto actual.  
  
 **Identificador de error:** BC30127  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Quite el atributo de la declaración o reemplácelo por un atributo que sea válido para el contexto.  
  
2.  Examine el mensaje de error y tome las medidas adecuadas.
