# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-using-the-ansi-auto-or-unicode-modifier"></a>La versión de destino de .NET Compact Framework no admite el uso del modificador Ansi, Auto o Unicode.
La versión de .NET Compact Framework con la que está trabajando no admite los modificadores `Ansi`, `Auto`y `Unicode` .  
  
 **Identificador de error:** BC30763  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite el modificador.
