# <a name="evaluation-of-expression-or-statement-stopped"></a>Se detuvo la evaluación de la expresión o instrucción
Se ha seleccionado el elemento de menú **Detener evaluación** , lo que ha provocado la detención de la evaluación.  
  
 **Identificador de error:** BC30721  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Permita que la ejecución continúe sin seleccionar **Detener evaluación**.
