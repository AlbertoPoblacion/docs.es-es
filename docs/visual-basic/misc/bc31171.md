# <a name="character-sequence----is-not-allowed-in-an-xml-comment"></a>No se permite la secuencia de caracteres '--' en un comentario XML.
Un literal de comentario XML contiene la secuencia de caracteres '--', que no se admite.  
  
 **Identificador de error:** BC31171  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite la secuencia de caracteres '--' del literal de comentario XML.
