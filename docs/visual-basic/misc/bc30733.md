# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>'\<membername >' ya se declaró por '\<NombreDeProcedimiento >', que se generó para este \<containername >

Se declaró un miembro que tiene el mismo nombre que un miembro que se declara mediante common language runtime.

**Identificador de error:** BC30733

## <a name="to-correct-this-error"></a>Para corregir este error

- Cambie el nombre del miembro para quitar el conflicto.
