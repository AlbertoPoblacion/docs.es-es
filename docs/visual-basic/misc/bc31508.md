# <a name="cannot-find-net-framework-directory-error"></a>No se puede encontrar el directorio de .NET Framework: \<error >
El directorio donde se almacenan los ensamblados [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] no se pudo encontrar.  
  
 **Identificador de error:** BC31508  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Vuelva a compilar el programa para ver si el error se repite.  
  
2.  Si el error continúa, guarde su trabajo y reinicie Visual Studio.  
  
3.  Si el error persiste, vuelva a Visual Basic.  
  
4.  Si el error persiste después de la reinstalación, informe al respecto a los Servicios de soporte técnico de Microsoft.
