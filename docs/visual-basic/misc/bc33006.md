# <a name="end-operator-must-be-the-first-statement-on-a-line"></a>'End Operator' debe ser la primera instrucción de una línea
Una instrucción `End Operator` sigue a un separador de instrucciones de dos puntos (`:`). `End Operator` debe ser la única instrucción en su línea de código fuente.  
  
 **Identificador de error:** BC33006  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Separe las distintas instrucciones en líneas diferentes.
