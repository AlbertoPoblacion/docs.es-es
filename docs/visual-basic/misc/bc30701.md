# <a name="nothing-cannot-be-evaluated"></a>No se puede evaluar 'Nothing'
Se usó la palabra clave `Nothing` en la ventana de depuración.  
  
 **Identificador de error:** BC30701  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   No use `Nothing` durante la depuración.
