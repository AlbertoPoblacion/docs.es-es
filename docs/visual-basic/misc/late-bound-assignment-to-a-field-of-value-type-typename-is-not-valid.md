# <a name="late-bound-assignment-to-a-field-of-value-type-typename-is-not-valid-when-name-is-the-result-of-a-late-bound-expression"></a>Asignación de tiempo de ejecución a un campo de tipo de valor '\<typename >' no es válida cuando '\<nombre >' es el resultado de una expresión enlazada en tiempo de ejecución
Intentó realizar una asignación enlazada en tiempo de ejecución que no es válida.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Realice la asignación en tiempo de compilación.
