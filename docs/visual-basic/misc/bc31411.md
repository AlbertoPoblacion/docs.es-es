# <a name="typename-must-be-declared-mustinherit-because-it-contains-methods-declared-mustoverride"></a>'\<typename >' debe declarar como 'MustInherit' porque contiene métodos declarados como 'MustOverride'
Los tipos que contienen miembros `MustOverride` deben declararse como `MustInherit`.  
  
 **Identificador de error:** BC31411  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Agregue el modificador `MustInherit` al tipo.
