# <a name="constructors-cannot-implement-interface-methods"></a>Los constructores no pueden implementar métodos de interfaz.
Ha intentado usar la palabra clave `New` para implementar un método de interfaz, lo que no es válido.  
  
 **Identificador de error:** BC30550  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Implemente interfaces con la palabra clave `Implements` .
