# <a name="withevents-variable-name-length-cannot-exceed-1019-characters"></a>La longitud del nombre de la variable WithEvents no puede superar los 1019 caracteres
El nombre o el identificador de cada variable declarada mediante `WithEvents` se limita a 1019 caracteres.  
  
 **Identificador de error:** BC32205  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Reduzca la longitud del identificador.
