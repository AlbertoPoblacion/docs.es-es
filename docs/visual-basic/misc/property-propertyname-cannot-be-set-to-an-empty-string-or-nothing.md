# <a name="property-propertyname-cannot-be-set-to-an-empty-string-or-nothing"></a>Propiedad \<propertyname > no puede establecerse en una cadena vacía o Nothing
La propiedad requiere un valor distinto de una cadena vacía ("") o `Nothing`. Esto puede ser el resultado del cálculo incorrecto del valor.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Proporcione un valor válido.
