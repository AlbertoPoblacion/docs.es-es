# <a name="interfacename1-cannot-implement-methodname-because-there-is-no-matching-method-on-interface-interfacename2"></a>'\<interfacename1 > ' no puede implementar\<' MethodName > ' porque no hay ningún \<método coincidente > en la\<interfaz ' interfacename2 ' > '
Intentó implementar un método en una interfaz cuando no hay ningún método coincidente en la interfaz de la que se derivó la primera interfaz.  
  
 **IDENTIFICADOR de error:** BC30401  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Asegúrese de haber escrito correctamente el nombre del método o proporcione el método que falta.
