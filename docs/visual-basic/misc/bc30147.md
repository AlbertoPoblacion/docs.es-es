# <a name="return-statements-are-not-valid-in-the-immediate-window"></a>Las instrucciones 'Return' no son válidas en la ventana Inmediato
La instrucción `Return` realiza una bifurcación y no está permitida en un contexto de depuración.  
  
 **Identificador de error:** BC30147  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- No emita una instrucción `Return` instrucción en la ventana **Inmediato** .
