# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>'\<qualifiedmethod > ' no es accesible en este contexto porque es '\<Modifier > '
Ha intentado obtener acceso a un método que no es accesible en este contexto debido a un modificador que se le ha aplicado.  
  
 **IDENTIFICADOR de error:** BC30390  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite el modificador.
