# <a name="xml-dtds-are-not-supported"></a>No se admiten las DTD XML.
Se ha especificado una definición de tipo de documento (DTD) como un literal XML.  
  
 **Identificador de error:** BC31175  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Elimine la DTD. Use la definición de esquemas XML (XSD) para especificar el esquema para un documento XML.
