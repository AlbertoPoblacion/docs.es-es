# <a name="notinheritable-visual-basic"></a>NotInheritable (Visual Basic)
Especifica que no se puede usar una clase como clase base.  
  
## <a name="remarks"></a>Comentarios  
 **Alternar términos**. A veces se denomina una clase que no se puede heredar un *sealed* clase.  
  
 El `NotInheritable` modificador se puede usar en este contexto:
