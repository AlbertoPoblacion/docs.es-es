# <a name="property-array-index-is-not-valid"></a>El índice de la matriz de propiedades no es válido
El valor proporcionado no es válido para un índice de matriz de propiedades.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe la documentación del componente para asegurarse de que el índice está dentro del intervalo válido para la propiedad especificada.
