# <a name="file-name-or-class-name-not-found-during-automation-operation-visual-basic"></a>No se encontró el nombre del archivo o de la clase durante la operación de automatización (Visual Basic)
El nombre especificado para el nombre de archivo o una clase en una llamada a la `GetObject` no se encontró la función.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Compruebe los nombres e inténtelo de nuevo. Asegúrese de que el nombre usado para el `class` parámetro coincide con que se registran con el sistema.
