# <a name="permission-denied-visual-basic"></a>Permiso denegado (Visual Basic)
Se ha intentado escribir en un disco protegido contra escritura o tener acceso a un archivo bloqueado.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Para abrir un archivo protegido contra escritura, cambie el atributo de protección contra escritura del archivo.  
  
2.  Asegúrese de que otro proceso no ha bloqueado el archivo y espere para abrir el archivo hasta que el otro proceso lo libere.  
  
3.  Para obtener acceso al registro, compruebe que los permisos de usuario incluyen este tipo de acceso al registro.
