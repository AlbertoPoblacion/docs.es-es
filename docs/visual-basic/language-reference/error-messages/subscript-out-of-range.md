# <a name="subscript-out-of-range-visual-basic"></a>El subíndice está fuera del intervalo (Visual Basic)
Un subíndice de matriz no es válido porque está fuera del intervalo permitido. El valor de subíndice más bajo de una dimensión siempre es 0 y el valor de subíndice más alto es devuelto por la `GetUpperBound` método para esa dimensión.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie el subíndice para que esté dentro del intervalo válido.
