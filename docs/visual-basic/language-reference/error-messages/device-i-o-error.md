# <a name="device-io-error"></a>Error de E/S del dispositivo
Se produjo un error de entrada o salido mientras el programa estaba usando un dispositivo como una impresora o unidad de disco.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Asegúrese de que el dispositivo está funcionando correctamente y, a continuación, vuelva a intentar la operación.
