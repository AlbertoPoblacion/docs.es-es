# <a name="some-subkeys-cannot-be-deleted"></a>Algunas subclaves no se pueden eliminar
Se ha intentado eliminar una clave del registro, pero la operación no se pudo porque algunas subclaves no se puede eliminar. Normalmente, esto es debido a una falta de permisos.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Asegúrese de que tiene permisos suficientes para eliminar las subclaves especificadas.
