# <a name="type-mismatch-visual-basic"></a>No coinciden los tipos (Visual Basic)
Se intentó convertir un valor a otro tipo de forma que no es válida.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Compruebe la asignación para asegurarse de que es válido.  
  
2. Asegúrese de que no ha pasado un objeto a un procedimiento que espera una sola propiedad o valor.  
  
3. Asegúrese de que no ha usado un nombre de proyecto o módulo donde se esperaba una expresión.
