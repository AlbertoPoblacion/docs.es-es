# <a name="compiler-error-cs0016"></a>Error del compilador CS0016

No se puede escribir en el archivo de salida 'file' — 'reason'

 El compilador no ha podido escribir en un archivo de salida. Compruebe la ruta al archivo para asegurarse de que existe. Si un archivo compilado anteriormente ya está en la ubicación, asegúrese de que se pueda escribir y que ningún proceso tenga actualmente un bloqueo en el archivo. Por ejemplo, asegúrese de que su archivo ejecutable no se está ejecutando cuando intenta realizar la compilación.
