# <a name="compiler-warning-level-1-cs1598"></a>Advertencia del compilador (nivel 1) CS1598
El analizador XML no se puede cargar por la siguiente razón: 'reason'. El archivo de documentación XML 'file' no se generará.  
  
 Se ha especificado la opción [-doc](../compiler-options/doc-compiler-option.md), pero el compilador no ha podido encontrar y cargar msxml3.dll. Asegúrese de que el archivo msxml3.dll está instalado y registrado.
