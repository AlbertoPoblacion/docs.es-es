# <a name="compiler-warning-level-2-cs0108"></a>Advertencia del compilador (nivel 2) CS0108

'member1' oculta el miembro heredado 'member2'. Use la palabra clave New si la ocultación estaba prevista.  
  
 Se ha declarado una variable con el mismo nombre como una variable en una clase base. En cambio, la palabra clave [new](../../../csharp/language-reference/keywords/new-modifier.md) no se ha usado. Esta advertencia le informa de que debería usar **new**; la variable se declara como si **new** se hubiera usado en la declaración.  
  
 En el ejemplo siguiente se genera el error CS0108:  

```csharp
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
