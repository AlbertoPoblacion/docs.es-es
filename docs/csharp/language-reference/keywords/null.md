# <a name="null-c-reference"></a>null (Referencia de C#)

La palabra clave `null` es un literal que representa una referencia nula que no hace referencia a ningún objeto. `null` es el valor predeterminado de las variables de tipo de referencia. Los tipos de valor normales no pueden ser NULL, excepto los [tipos de valor que admiten un valor NULL](../builtin-types/nullable-value-types.md).
