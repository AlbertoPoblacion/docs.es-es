# <a name="null-c-reference"></a>null (Referencia de C#)

La palabra clave `null` es un literal que representa una referencia nula que no hace referencia a ningún objeto. `null` es el valor predeterminado de las variables de tipo de referencia. Los tipos de valor normales no pueden ser nulos, pero en C# 2.0 se han introducido tipos de valor que aceptan valores NULL. Vea [Tipos que aceptan valores NULL](../../programming-guide/nullable-types/index.md).
