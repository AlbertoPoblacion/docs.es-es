# <a name="ascending-c-reference"></a>ascending (Referencia de C#)
La palabra clave contextual `ascending` se usa en la [cláusula orderby](./orderby-clause.md) en expresiones de consulta para especificar que el criterio de ordenación es de menor a mayor. Como `ascending` es el criterio de ordenación predeterminado, no tiene que especificarlo.
