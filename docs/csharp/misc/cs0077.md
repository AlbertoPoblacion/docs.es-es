# <a name="compiler-error-cs0077"></a>Error del compilador CS0077
El operador as se debe usar con un tipo de referencia o un tipo que acepta valores NULL ('int' es un tipo de valor que no acepta valores NULL)  
  
 Se pasó al operador [as](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#as-operator) un [tipo de valor](../../csharp/language-reference/keywords/value-types.md). Puesto que `as` puede devolver [null](../../csharp/language-reference/keywords/null.md), solo se le pueden pasar [tipos de referencia](../../csharp/language-reference/keywords/reference-types.md) o un tipo que acepta valores NULL. Para obtener más información acerca de los tipos que aceptan valores NULL, vea [tipos que aceptan valores NULL](../../csharp/programming-guide/nullable-types/index.md).  
  
 El ejemplo siguiente genera la advertencia CS0077:  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
