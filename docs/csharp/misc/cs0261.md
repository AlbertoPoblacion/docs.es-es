# <a name="compiler-error-cs0261"></a>Error del compilador CS0261
Las declaraciones parciales de 'type' deben ser todas clases, todas estructuras o todas interfaces  
  
 Este error se produce si un tipo parcial se declara como un tipo diferente de construcción en distintos lugares. Para más información, vea [Clases y métodos parciales](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).  
  
 El ejemplo siguiente genera la advertencia CS0261:  
  
```csharp  
// CS0261.cs  
partial class A  // CS0261 – A declared as a class here, but as a struct below  
{  
}  
  
partial struct A  
{  
}  
```
