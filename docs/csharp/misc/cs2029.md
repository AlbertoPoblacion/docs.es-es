# <a name="compiler-warning-level-1-cs2029"></a>Advertencia del compilador (nivel 1) CS2029
Valor no válido para '/define'; 'identificador' no es un identificador válido.  
  
 Esta advertencia se produce si el valor que se usa en la opción [/define](../language-reference/compiler-options/define-compiler-option.md) tiene algunos caracteres no válidos.  
  
 No se puede suprimir esta advertencia con la opción [/nowarn](../language-reference/compiler-options/nowarn-compiler-option.md) .
