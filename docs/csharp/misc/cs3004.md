# <a name="compiler-warning-level-1-cs3004"></a>Advertencia del compilador (nivel 1) CS3004
Los caracteres Unicode mixtos y descompuestos no son conformes a CLS  
  
 Solo se permiten caracteres UNICODE compuestos en los identificadores [public](../../csharp/language-reference/keywords/public.md), [protected](../../csharp/language-reference/keywords/protected.md)o `protected internal` para que sean conformes a Common Language Specification (CLS). Para obtener más información sobre la conformidad con CLS, vea [independencia del lenguaje y componentes independientes del lenguaje](../../standard/language-independence.md).
