# <a name="compiler-error-cs0588"></a>Error del compilador CS0588

Error interno del compilador: fase 'LEX'

 Intente determinar si se están produciendo errores en el compilador porque no puede analizar una sintaxis inesperada. Si no es así, [hable con nosotros](/visualstudio/ide/talk-to-us).
 
> [!NOTE]
> Este error del compilador ya no se usa en Roslyn.
