# <a name="compiler-error-cs0272"></a>Error del compilador CS0272
La propiedad o el indexador 'property/indexer' no se puede usar en este contexto porque el descriptor de acceso set es inaccesible  
  
 Este error se produce cuando el descriptor de acceso `set` no es accesible para el c칩digo de programa. Para resolver este error, aumente la accesibilidad del descriptor de acceso o cambie la ubicaci칩n de la llamada. ## <a name="example"></a>Ejemplo  
 El siguiente ejemplo genera el error CS0272:  
  
```csharp  
// CS0272.cs  
public class MyClass  
{  
    public int Property  
    {  
        get { return 0; }  
        private set { }  
    }  
}  
  
public class Test  
{  
    static void Main()  
    {  
        MyClass c = new MyClass();  
        c.Property = 10;      // CS0272  
        // To resolve, remove the previous line   
        // or use an appropriate modifier on the set accessor.  
    }  
}  
```
