# <a name="compiler-error-cs0037"></a>Error del compilador CS0037
No se puede convertir null a 'type' porque es un tipo de valor que no acepta valores Null  
  
 El compilador no puede asignar null a un tipo de valor; null sólo puede asignarse a un [tipo de referencia](../language-reference/keywords/reference-types.md) o a un tipo que acepte valores Null. [struct](../language-reference/keywords/struct.md) es un tipo de valor. Para más información, vea [Tipos que aceptan valores NULL](../programming-guide/nullable-types/index.md).  
  
 El ejemplo siguiente genera la advertencia CS0037:  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
