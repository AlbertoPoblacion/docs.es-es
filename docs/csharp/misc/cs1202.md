# <a name="compiler-warning-level-1-cs1202"></a>Advertencia del compilador (nivel 1) CS1202
La característica 'invalid feature' está en desuso. Use 'valid feature' en su lugar.  
  
 La característica que intenta usar está en desuso. Actualice el código para usar la característica válida en su lugar.
