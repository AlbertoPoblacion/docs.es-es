# <a name="compiler-warning-level-4-cs0422"></a>Advertencia del compilador (nivel 4) CS0422

La opción /incremental ya no se admite

 La compilación incremental (**/incr** o **/incremental**) no se admite en C- 2.0.

> [!NOTE]
> Esta advertencia del compilador ya no se utiliza en Roslyn.
