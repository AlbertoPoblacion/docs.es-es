# <a name="compiler-error-cs0167"></a>Error del compilador CS0167
Falta el método Invoke en el delegado 'delegado'.  
  
 Ha importado y usado un programa administrado (uno que usa .NET Framework Common Language Runtime) que se creó con otro compilador. Ese compilador permitía un [delegado](../../csharp/language-reference/keywords/delegate.md)formado incorrectamente. Por lo tanto, el método `Invoke` no estaba disponible. Para obtener más información, vea [Delegados (Guía de programación de C#)](../../csharp/programming-guide/delegates/index.md).
