# <a name="compiler-error-cs0242"></a>Error del compilador CS0242
La operación en cuestión no está definida en punteros void  
  
 No se permite incrementar un puntero void. Para obtener más información, vea [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md) (Código no seguro y punteros [Guía de programación de C#]).  
  
 El ejemplo siguiente genera la advertencia CS0242:  
  
```csharp  
// CS0242.cs  
// compile with: /unsafe  
class TestClass  
{  
   public unsafe void Test()  
   {  
      void * p = null;  
      p++;   // CS0242, incrementing a void pointer not allowed  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
