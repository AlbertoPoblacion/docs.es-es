# <a name="compiler-error-cs0267"></a>Error del compilador CS0267
El modificador 'partial' solo puede aparecer inmediatamente antes de 'class', 'struct' o 'interface'  
  
 La colocación del modificador **partial** era incorrecta en la declaración de la clase, el struct o la interfaz. Para corregir el error, reordene los modificadores. Para más información, vea [Clases y métodos parciales](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).  
  
 El ejemplo siguiente genera la advertencia CS0267:  
  
```csharp  
// CS0267.cs  
public partial class MyClass  
{  
   public MyClass()  
   {  
   }  
}  
  
partial public class MyClass  // CS0267  
// Try this line instead:  
// public partial class MyClass  
{  
   public void Foo()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
