# <a name="compiler-error-cs1551"></a>Error del compilador CS1551
Los indizadores deben tener al menos un parámetro  
  
 Se declaró un [indizador](../programming-guide/indexers/index.md) que no toma ningún argumento.  
  
 El ejemplo siguiente genera la advertencia CS1551:  
  
```csharp  
// CS1551.cs  
public class MyClass  
{  
   int intI;  
  
   int this[]   // CS1551  
   // try the following line instead  
   // int this[int i]  
   {  
      get  
      {  
         return intI;  
      }  
      set  
      {  
         intI = value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
