# <a name="compiler-warning-level-4-cs0109"></a>Advertencia del compilador (nivel 4) CS0109
El miembro 'miembro' no oculta a un miembro heredado. No se necesita la palabra clave new.  
  
 Una declaración de clase incluía la palabra clave [new](../../csharp/language-reference/keywords/new-modifier.md) aunque la declaración no reemplaza una declaración existente en una clase base. Puede eliminar la palabra clave **new** .  
  
 El ejemplo siguiente genera la advertencia CS0109:  
  
```csharp  
// CS0109.cs  
// compile with: /W:4  
namespace x  
{  
   public class a  
   {  
      public int i;  
   }  
  
   public class b : a  
   {  
      public new int i;  
      public new int j;   // CS0109  
      public static void Main()  
      {  
      }  
   }  
}  
```
