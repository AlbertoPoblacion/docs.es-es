# <a name="compiler-error-cs1553"></a>Error del compilador CS1553
Declaración no es válida; Use ' operador modificador \<dest-type > (...' en lugar de  
  
 El tipo de valor devuelto para un [operador de conversión](../../csharp/language-reference/keywords/implicit.md) debe preceder inmediatamente a la lista de parámetros y *modificador* sea `implicit` o **explícita**.  
  
 El ejemplo siguiente genera la advertencia CS1553:  
  
```csharp  
// CS1553.cs  
class MyClass  
{  
   public static int implicit operator (MyClass f)   // CS1553  
   // try the following line instead  
   // public static implicit operator int (MyClass f)  
   {  
      return 6;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
