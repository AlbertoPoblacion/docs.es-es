# <a name="compiler-error-cs2012"></a>Error del compilador CS2012
No se puede abrir 'label' para escritura  
  
 Al usar la opción del compilador [-bugreport (:](../language-reference/compiler-options/bugreport-compiler-option.md) `file`, no se pudo abrir el archivo para escribir en él. Asegúrese de que ha especificado un nombre de archivo válido y de que el archivo no es de solo lectura.
