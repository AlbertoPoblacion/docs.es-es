# <a name="compiler-error-cs0104"></a>Error del compilador CS0104

 El programa contiene directivas [using](../language-reference/keywords/using.md) para dos espacios de nombres y el c√≥digo hace referencia a un nombre que aparece en ambos.

```csharp  
// CS0104.cs  
using x;  
using y;  
  
namespace x  
{  
   public class Test  
   {  
   }

{  
   public class Test  
   {  
   }  
}
```
