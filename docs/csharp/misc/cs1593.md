# <a name="compiler-error-cs1593"></a>Error del compilador CS1593
El delegado 'del' no toma argumentos 'number'  
  
 El número de argumentos pasados a una invocación [delegate](../language-reference/keywords/delegate.md) no coincide con el número de parámetros de la declaración de delegado.  
  
 El ejemplo siguiente genera la advertencia CS1593:  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
