# <a name="compiler-error-cs0263"></a>Error del compilador CS0263
Las declaraciones parciales de 'tipo' no deben especificar clases base diferentes.  
  
 Al definir un tipo en declaraciones parciales, especifique los mismos tipos bases en cada una de las declaraciones parciales. Para más información, vea [Clases y métodos parciales](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).  
  
 El ejemplo siguiente genera la advertencia CS0263:  
  
```csharp  
// CS0263.cs  
// compile with: /target:library  
class B1  
{  
}  
  
class B2  
{  
}  
partial class C : B1  // CS0263 – is the base class B1 or B2?  
{  
}  
  
partial class C : B2  
{  
}  
```
