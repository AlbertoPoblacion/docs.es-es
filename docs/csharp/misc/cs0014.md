# <a name="compiler-error-cs0014"></a>Error del compilador CS0014
No se pudo encontrar el archivo necesario 'archivo'.  
  
 Un archivo que es necesario para el compilador no se encuentra en el sistema. Asegúrese de que la ruta de acceso es correcta. Si el archivo es un archivo de sistema de Visual Studio, deberá reparar la instalación, o bien quitar y reinstalar Visual Studio completamente.
