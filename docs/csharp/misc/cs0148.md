# <a name="compiler-error-cs0148"></a>Error del compilador CS0148
El delegado 'delegate' no tiene un constructor válido  
  
 Ha importado y usado un programa administrado (uno que usa .NET Framework Common Language Runtime) que se creó con otro compilador. Ese compilador permitía un constructor [delegate](../language-reference/keywords/delegate.md) formado incorrectamente. Para obtener más información, vea [Delegados (Guía de programación de C#)](../programming-guide/delegates/index.md).
