# <a name="compiler-error-cs0505"></a>Error del compilador CS0505
'miembro1': no se puede reemplazar porque 'miembro2' no es una función.  
  
 Una declaración de clase intentó reemplazar un elemento que no es un método en una clase base. Los reemplazos deben coincidir en el tipo de miembro. Si quiere un método con el mismo nombre que un método en una clase base, use [new](../../csharp/language-reference/keywords/new-modifier.md) (y no [override](../../csharp/language-reference/keywords/override.md)) en la declaración del método en la clase base.  
  
 El ejemplo siguiente genera la advertencia CS0505:  
  
```csharp  
// CS0505.cs  
// compile with: /target:library  
public class clx  
{  
   public int i;  
}  
  
public class cly : clx  
{  
   public override int i() { return 0; }   // CS0505  
}  
```
