# <a name="compiler-error-cs0160"></a>Error del compilador CS0160
Una cláusula catch previa ya detecta todas las excepciones de este tipo o de tipo superior ('tipo')  
  
Una serie de instrucciones [catch](../../csharp/language-reference/keywords/try-catch.md) debe estar en orden de derivación decreciente. Por ejemplo, los objetos más derivados deben aparecer primero.
  
 Para obtener más información, consulte [excepciones y control de excepciones](../../csharp/programming-guide/exceptions/index.md).  
  
 El ejemplo siguiente genera la advertencia CS0160:  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
