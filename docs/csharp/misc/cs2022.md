# <a name="compiler-error-cs2022"></a>Error del compilador CS2022
Las opciones '/out' y '/target' deben aparecer antes de los nombres de archivo de código fuente  
  
 Las opciones del compilador [/out (establecer el nombre de archivo de salida)](../../csharp/language-reference/compiler-options/out-compiler-option.md) y [/target (especificar el formato del archivo de salida)](../../csharp/language-reference/compiler-options/target-compiler-option.md) , cuando se especifican en la línea de comandos, deben preceder a los archivos de código fuente.
