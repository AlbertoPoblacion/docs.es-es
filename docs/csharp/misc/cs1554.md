# <a name="compiler-error-cs1554"></a>Error del compilador CS1554
Declaración no es válida; Use '\<tipo > op operador (...' en lugar de  
  
 El tipo de valor devuelto para un [operador](../../csharp/language-reference/keywords/operator.md) definido por el usuario debe aparecer antes del operador de palabra clave.  
  
 El ejemplo siguiente genera la advertencia CS1554:  
  
```csharp  
// CS1554.cs  
class MyClass  
{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
