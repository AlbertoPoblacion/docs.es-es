# <a name="compiler-error-cs1020"></a>Error del compilador CS1020
Se esperaba un operador binario sobrecargable  
  
 Se ha intentado definir una sobrecarga del operador, pero el operador no era un operador binario sobrecargable, que toma dos parámetros. Para obtener la lista de operadores sobrecargables, consulte el [operadores sobrecargables](../../csharp/language-reference/operators/operator-overloading.md#overloadable-operators) sección de la [la sobrecarga de operadores](../../csharp/language-reference/operators/operator-overloading.md) artículo.
  
 El ejemplo siguiente genera la advertencia CS1020:  
  
```csharp  
// CS1020.cs  
public class iii  
{  
   public static int operator ++(iii aa, int bb)   // CS1020, change ++ to +  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
