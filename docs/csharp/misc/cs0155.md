# <a name="compiler-error-cs0155"></a>Error del compilador CS0155
El tipo detectado o producido debe derivarse de System.Exception  
  
 Se intentó pasar un tipo de datos que no se deriva de **System.Exception** en un bloque [catch](../../csharp/language-reference/keywords/try-catch.md) . Solo los tipos de datos que se derivan de **System.Exception** se pueden pasar a un bloque **catch** . Para obtener más información, consulte [excepciones y control de excepciones](../../csharp/programming-guide/exceptions/index.md).  
  
 El ejemplo siguiente genera la advertencia CS0155:  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
