# <a name="compiler-error-cs2003"></a>Error del compilador CS2003
El archivo de respuesta 'archivo' se incluyó varias veces.  
  
 Se pasó al compilador un archivo de respuesta más de una vez. Un archivo de respuesta solo puede pasarse al compilador una vez por archivo de salida.  
  
 Para obtener más información sobre los archivos de respuesta, vea [@ (especificar archivo de respuesta)](../../csharp/language-reference/compiler-options/response-file-compiler-option.md).
