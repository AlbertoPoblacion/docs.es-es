# <a name="compiler-error-cs1681"></a>Error del compilador CS1681
No se puede definir de nuevo el alias externo global  
  
 El alias global ya est√° definido para incluir todas las referencias sin alias y, por lo tanto, no se puede volver a definir.  
  
## <a name="example"></a>Ejemplo  
 El ejemplo siguiente genera la advertencia CS1681.  
  
```csharp  
// CS1681.cs  
// compile with: /reference:global=System.dll  
// CS1681 expected  
  
// try this instead: /reference:System.dll  
class A  
{  
   static void Main() {}  
}  
```
