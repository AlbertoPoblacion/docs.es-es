# <a name="compiler-error-cs1557"></a>Error del compilador CS1557
No se puede usar 'class' para el método Main porque se encuentra en un archivo de salida diferente  
  
 Se ha especificado la opción del compilador [-Main](../language-reference/compiler-options/main-compiler-option.md) para un archivo de salida en una compilación de archivos de múltiples salidas. Sin embargo, no se encontró la clase en el código fuente de la compilación /main; se encontró un archivo de código fuente de uno de los otros archivos de salida de la compilación.
