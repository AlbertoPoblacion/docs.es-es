# <a name="compiler-error-cs0594"></a>Error del compilador CS0594
La constante de punto flotante está fuera del intervalo del tipo 'type'  
  
 Se asignó un valor a una variable de punto flotante que es demasiado grande para las variables de este tipo de datos. Consulte [Tabla de tipos integrales](../../csharp/language-reference/builtin-types/integral-numeric-types.md) para obtener información sobre el intervalo de valores permitido en los tipos de datos.  
  
 El ejemplo siguiente genera la advertencia CS0594:  
  
```csharp  
// CS0594.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         float f = 6.77777777777E400;   // CS0594, value too large  
      }  
   }  
}  
```
