# <a name="compiler-error-cs1647"></a>Error del compilador CS1647
Una expresión es demasiado larga o compleja para compilar cerca de 'code'  
  
 Se produjo un desbordamiento de pila en el compilador que procesa el código. Para resolver este error, simplifique el código. Si el código es válido, obtener ayuda en la [Comunidad de desarrolladores](https://developercommunity.visualstudio.com/spaces/61/index.html) sitio o [StackOverflow](https://stackoverflow.com/).
