# <a name="compiler-error-cs0076"></a>Error del compilador CS0076
El nombre de enumerador 'value__' está reservado y no se puede usar.  
  
 Un elemento de una [enumeración](../language-reference/builtin-types/enum.md) no puede tener un identificador denominado **value__** .
