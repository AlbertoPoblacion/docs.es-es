# <a name="compiler-error-cs0568"></a>Error del compilador CS0568
Los structs no pueden contener constructores explícitos sin parámetros  
  
 Cada [struct](../language-reference/keywords/struct.md) ya tiene un constructor sin parámetros que inicializa el objeto a cero. Por lo tanto, los constructores que se pueden crear para un struct deben tener uno o varios parámetros.  
  
 El ejemplo siguiente genera la advertencia CS0568:  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
