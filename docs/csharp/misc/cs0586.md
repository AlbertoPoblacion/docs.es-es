# <a name="compiler-error-cs0586"></a>Error del compilador CS0586
Error interno del compilador: fase 'stage'  
  
 Intente determinar si el compilador está fallando porque no puede analizar una sintaxis inesperada. Si no es así, intente [hable con nosotros](/visualstudio/ide/talk-to-us).
