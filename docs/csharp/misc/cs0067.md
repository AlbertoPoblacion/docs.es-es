# <a name="compiler-warning-level-3-cs0067"></a>Advertencia del compilador (nivel 3) CS0067
Nunca se usa el evento 'event'.  
  
 Se ha declarado un [evento](../language-reference/keywords/event.md) , pero nunca se us√≥ en la clase en la que se ha declarado.  
  
 El ejemplo siguiente genera la advertencia CS0067:  
  
```csharp  
// CS0067.cs  
// compile with: /W:3  
using System;  
delegate void MyDelegate();  
  
class MyClass  
{  
   public event MyDelegate evt;   // CS0067

   {  
   }  
}
```
