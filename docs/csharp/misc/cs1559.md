# <a name="compiler-error-cs1559"></a>Error del compilador CS1559
No se puede usar 'object' para el método Main porque se ha importado  
  
 Se especificó una clase no válida para la opción del compilador [-Main](../language-reference/compiler-options/main-compiler-option.md) ; no se puede usar la clase como ubicación para el método [Main](../programming-guide/main-and-command-args/index.md) .
