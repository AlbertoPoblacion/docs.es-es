# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>Modernización de las aplicaciones .NET existentes en aplicaciones optimizadas para la nube

> Visión: Modernice las aplicaciones .NET Framework existentes en aplicaciones optimizadas para la nube, a fin de mejorar significativamente la agilidad del desarrollo, para una entrega más rápida y costos de entrega de aplicaciones más bajos.

Para sacar provecho de las ventajas de la nube y las nuevas tecnologías, como los contenedores, al menos debe modernizar parcialmente las aplicaciones .NET existentes. A la larga, la modernización de las aplicaciones empresariales reducirá el costo total de propiedad.

La modernización parcial de una aplicación no conlleva necesariamente una migración completa ni un cambio de diseño. Puede modernizar las aplicaciones existentes por primera vez con un proceso de modernización importante, pero fácil de llevar a cabo. Puede mantener la base de código actual, escrita en las versiones de .NET Framework existentes, con cualquier dependencia de Windows y IIS. En la figura 4-1 se ilustra cómo las aplicaciones optimizadas para la nube se posicionan en los modelos de madurez de modernización de aplicaciones de Azure.
