# <a name="choosing-between-net-core-and-net-framework-for-docker-containers"></a>Selección entre .NET Core y .NET Framework para contenedores de Docker

Se admiten dos marcos para compilar aplicaciones de Docker en contenedor del lado servidor con .NET: [.NET Framework y .NET Core](https://www.microsoft.com/net/download). Ambos comparten muchos de los componentes de la plataforma .NET y es posible compartir código entre ellos. Aun así, presentan diferencias fundamentales, y la elección del marco dependerá de lo que quiera realizar. En esta sección se proporciona orientación sobre cuándo se debe elegir cada marco.
