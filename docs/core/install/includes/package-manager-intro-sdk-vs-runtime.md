Instale el SDK (que incluye el entorno de ejecución) si quiere desarrollar aplicaciones de .NET Core. O bien, si solo necesita ejecutar aplicaciones, instale el entorno de ejecución. Si va a instalar el entorno de ejecución, le recomendamos que instale el [entorno de ejecución de ASP.NET Core](#install-the-aspnet-core-runtime), ya que incluye los de .NET Core y ASP.NET Core.

Si ya ha instalado el SDK o el entorno de ejecución, use los comandos `dotnet --list-sdks` y `dotnet --list-runtimes` para ver qué versiones están instaladas. Para más información, consulte [Cómo comprobar que .NET Core ya está instalado](../how-to-detect-installed-versions.md?pivots=os-linux).
