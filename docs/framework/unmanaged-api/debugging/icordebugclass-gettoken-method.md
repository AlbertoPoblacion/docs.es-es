# <a name="icordebugclassgettoken-method"></a>ICorDebugClass::GetToken (Método)
Obtiene el token de metadatos de `TypeDef` que hace referencia a la definición de esta clase.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT GetToken (  
    [out] mdTypeDef          *pTypeDef  
);  
```  
  
## <a name="parameters"></a>Parámetros  
 `pTypeDef`  
 enuncia Un puntero a un token de `mdTypeDef` que hace referencia a la definición de esta clase.
