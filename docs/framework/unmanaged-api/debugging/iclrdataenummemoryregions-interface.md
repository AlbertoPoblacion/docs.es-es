# <a name="iclrdataenummemoryregions-interface"></a>ICLRDataEnumMemoryRegions (Interfaz)
Proporciona un método para enumerar las regiones de memoria especificadas por los llamadores.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[EnumMemoryRegions (método)](../../../../docs/framework/unmanaged-api/debugging/iclrdataenummemoryregions-enummemoryregions-method.md)|Enumera las áreas de memoria especificadas.|
