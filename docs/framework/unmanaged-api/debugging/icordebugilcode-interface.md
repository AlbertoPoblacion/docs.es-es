# <a name="icordebugilcode-interface"></a>ICorDebugILCode (Interfaz)
[Compatible con .NET Framework 4.5.2 y versiones posteriores]  
  
 Representa un segmento de código de lenguaje intermedio (IL).  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[GetEHClauses (método)](icordebugilcode-getehclauses-method.md)|Devuelve un puntero a una lista de cláusulas de control de excepciones (EH) definidas para este IL.|
