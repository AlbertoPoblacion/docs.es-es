# <a name="icordebugmoduleenum-interface"></a>ICorDebugModuleEnum (Interfaz)

Implementa los métodos ICorDebugEnum y enumera las matrices ICorDebugModule.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[Next (método)](../../../../docs/framework/unmanaged-api/debugging/icordebugmoduleenum-next-method.md)|Obtiene el número especificado de instancias de `ICorDebugModule` de la enumeración, comenzando en la posición actual.|  
  
## <a name="remarks"></a>Comentarios  
  
> [!NOTE]
> Esta interfaz no admite que se la llame de forma remota, ya sea entre procesos o entre equipos.
