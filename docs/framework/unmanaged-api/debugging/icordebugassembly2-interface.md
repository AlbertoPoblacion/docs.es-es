# <a name="icordebugassembly2-interface"></a>Interfaz ICorDebugAssembly2

Representa un ensamblado. Esta interfaz es una extensión de la interfaz ICorDebugAssembly.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[IsFullyTrusted (método)](icordebugassembly2-isfullytrusted-method.md)|Obtiene un valor que indica si el sistema de seguridad en tiempo de ejecución ha concedido plena confianza al ensamblado.|  
  
## <a name="remarks"></a>Notas  
  
> [!NOTE]
> Esta interfaz no admite que se la llame de forma remota, ya sea entre procesos o entre equipos.
