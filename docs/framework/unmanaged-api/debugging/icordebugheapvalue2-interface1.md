# <a name="icordebugheapvalue2-interface"></a>ICorDebugHeapValue2 (Interfaz)

Extensión de ICorDebugHeapValue que proporciona compatibilidad con los identificadores de Common Language Runtime (CLR).  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[CreateHandle (método)](../../../../docs/framework/unmanaged-api/debugging/icordebugheapvalue2-createhandle-method.md)|Crea un identificador del tipo especificado para este objeto `ICorDebugHeapValue2`.|  
  
## <a name="remarks"></a>Comentarios  
  
> [!NOTE]
> Esta interfaz no admite que se la llame de forma remota, ya sea entre procesos o entre equipos.
