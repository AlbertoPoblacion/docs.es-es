# <a name="icorpublishenumreset-method"></a>ICorPublishEnum::Reset (Método)
Mueve el cursor al principio de la enumeración.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
HRESULT Reset ();  
```  
  
## <a name="parameters"></a>Parámetros  
 `Reset` no toma ningún parámetro.
