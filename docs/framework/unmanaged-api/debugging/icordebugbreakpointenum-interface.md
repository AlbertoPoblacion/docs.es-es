# <a name="icordebugbreakpointenum-interface"></a>ICorDebugBreakpointEnum (Interfaz)

Implementa los métodos de ICorDebugEnum y enumera las matrices de ICorDebugBreakpoint.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[Next (método)](../../../../docs/framework/unmanaged-api/debugging/icordebugbreakpointenum-next-method.md)|Obtiene el número especificado de instancias de `ICorDebugBreakpoint` de la enumeración, comenzando en la posición actual.|  
  
## <a name="remarks"></a>Comentarios  
  
> [!NOTE]
> Esta interfaz no admite que se la llame de forma remota, ya sea entre procesos o entre equipos.
