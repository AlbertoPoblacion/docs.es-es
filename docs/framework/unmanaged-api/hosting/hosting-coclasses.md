# <a name="hosting-coclasses"></a>Coclases para el hospedaje
Esta sección describe las coclases no administradas de API que usa la API de hospedaje.  
  
## <a name="in-this-section"></a>En esta sección  
 [CLRRuntimeHost (coclase)](../../../../docs/framework/unmanaged-api/hosting/clrruntimehost-coclass.md)  
 Proporciona interfaces para administrar la ejecución de código por el tiempo de ejecución.  
  
 [ComCallUnmarshal (coclase)](../../../../docs/framework/unmanaged-api/hosting/comcallunmarshal-coclass.md)  
 Proporciona interfaces para administrar el cálculo de referencias de punteros de interfaz.  
  
 [CorRuntimeHost (coclase)](../../../../docs/framework/unmanaged-api/hosting/corruntimehost-coclass.md)  
 Proporciona interfaces para administrar las aplicaciones que se ejecutan por common language runtime.  
  
 [TypeNameFactory (coclase)](../../../../docs/framework/unmanaged-api/hosting/typenamefactory-coclass.md)  
 Proporciona una interfaz para administrar la anulación de la construcción de un nombre de tipo.
