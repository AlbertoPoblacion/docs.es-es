# <a name="ihostautoevent-interface"></a>IHostAutoEvent (Interfaz)
Proporciona una representación de la implementación del host de un evento de restablecimiento automático.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[Set (método)](../../../../docs/framework/unmanaged-api/hosting/ihostautoevent-set-method.md)|Establece la instancia de `IHostAutoEvent` actual en un estado señalado.|  
|[Wait (método)](../../../../docs/framework/unmanaged-api/hosting/ihostautoevent-wait-method.md)|Hace que la instancia de `IHostAutoEvent` actual espere hasta que el evento sea propiedad o transcurra un período de tiempo especificado.|
