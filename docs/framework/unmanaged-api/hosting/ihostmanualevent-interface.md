# <a name="ihostmanualevent-interface"></a>IHostManualEvent (Interfaz)
Proporciona la implementación del host de una representación de un evento de restablecimiento manual.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[Reset (método)](../../../../docs/framework/unmanaged-api/hosting/ihostmanualevent-reset-method.md)|Restablece la instancia de `IHostManualEvent` actual a un estado no señalado.|  
|[Set (método)](../../../../docs/framework/unmanaged-api/hosting/ihostmanualevent-set-method.md)|Establece la instancia de `IHostManualEvent` actual en un estado señalado.|  
|[Wait (método)](../../../../docs/framework/unmanaged-api/hosting/ihostmanualevent-wait-method.md)|Hace que la instancia de `IHostManualEvent` actual espere hasta que sea propiedad o transcurra un período de tiempo especificado.|
