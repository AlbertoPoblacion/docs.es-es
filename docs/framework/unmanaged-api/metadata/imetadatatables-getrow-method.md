# <a name="imetadatatablesgetrow-method"></a>IMetaDataTables::GetRow (Método)
Obtiene la fila en el índice de fila especificado de la tabla en el índice de tabla especificado.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT GetRow (   
    [in]  ULONG   ixTbl,  
    [in]  ULONG   rid,  
    [out] void    **ppRow  
);  
```  
  
## <a name="parameters"></a>Parámetros  
 `ixTbl`  
 de Índice de la tabla de la que se recuperará la fila.  
  
 `rid`  
 de Índice de la fila que se va a obtener.  
  
 `ppRow`  
 enuncia Un puntero a un puntero a la fila.  
  
## <a name="remarks"></a>Comentarios  
 No se recomienda el uso de este método, ya que no devuelve resultados coherentes. Para obtener información acerca de la tabla de GUID, consulte la documentación de Common Language Infrastructure (CLI), especialmente "partición II: definición y semántica de los metadatos".
