# <a name="imetadataimportresetenum-method"></a>IMetaDataImport::ResetEnum (Método)
Restablece el enumerador especificado a la posición especificada.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT ResetEnum (  
   [in] HCORENUM    hEnum,
   [in] ULONG       ulPos  
);  
```  
  
## <a name="parameters"></a>Parámetros  
 `hEnum`  
 [en] El enumerador que se ha de restablecer.  
  
 `ulPos`  
 [en] La nueva posición en la que colocar el enumerador.
