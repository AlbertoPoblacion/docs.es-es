# <a name="imetadatainfo-interface"></a>IMetaDataInfo (Interfaz)
Proporciona un método que obtiene información sobre la asignación de metadatos desde un archivo en disco a la memoria.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[Método GetFileMapping](../../../../docs/framework/unmanaged-api/metadata/imetadatainfo-getfilemapping-method.md)|Obtiene la región de memoria del archivo asignado y el tipo de asignación.|
