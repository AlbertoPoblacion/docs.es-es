# <a name="imetadatatablesgetguid-method"></a>IMetaDataTables::GetGuid (Método)
Gets a GUID from the row at the specified index.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT GetGuid (   
    [in]  ULONG       ixGuid,  
    [out] const GUID  **ppGUID  
);  
```  
  
## <a name="parameters"></a>Parámetros  
 `ixGuid`  
 [in] The index of the row from which to get the GUID.  
  
 `ppGuid`  
 [out] A pointer to a pointer to the GUID.  
  
## <a name="remarks"></a>Comentarios  
 We do not recommend the use of this method, because it does not return consistent results. For information about the GUID table, see the Common Language Infrastructure (CLI) documentation, especially "Partition II: Metadata Definition and Semantics".
