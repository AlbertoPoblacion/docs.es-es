# <a name="imetadataemitdefinemethodimpl-method"></a>IMetaDataEmit::DefineMethodImpl (Método)
Crea una definición para la implementación de un método heredado de una interfaz y devuelve un token a esa definición de implementación de método.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT DefineMethodImpl (
    [in]  mdTypeDef         td,
    [in]  mdToken           tkBody,
    [in]  mdToken           tkDecl  
);  
```  
  
## <a name="parameters"></a>Parámetros  
 `td`  
 [en] El `mdTypedef` token de la clase de implementación.  
  
 `tkBody`  
 [en] El `mdMethodDef` `mdMemberRef` token o del cuerpo del código.  
  
 `tkDecl`  
 [en] El `mdMethodDef` `mdMemberRef` token o del método de interfaz que se está implementando.
