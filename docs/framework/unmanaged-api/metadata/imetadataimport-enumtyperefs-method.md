# <a name="imetadataimportenumtyperefs-method"></a>IMetaDataImport::EnumTypeRefs (Método)
Enumera los tokens de TypeRef definidos en el ámbito de metadatos actual.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT EnumTypeRefs (  
   [in, out] HCORENUM    *phEnum,
   [out] mdTypeRef       rTypeRefs[],  
   [in]  ULONG           cMax,
   [out] ULONG           *pcTypeRefs  
);  
```  
  
## <a name="parameters"></a>Parámetros  
 `phEnum`  
 [adentro, fuera] Un puntero al enumerador. Debe ser NULL para la primera llamada de este método.  
  
 `rTypeRefs`  
 [fuera] Matriz utilizada para almacenar los tokens TypeRef.  
  
 `cMax`  
 [in] Tamaño máximo de la matriz `rTypeRefs`.  
  
 `pcTypeRefs`  
 [fuera] Puntero al número de tokens TypeRef devueltos en `rTypeRefs`.  
  
## <a name="return-value"></a>Valor devuelto  
  
|HRESULT|Descripción|  
|-------------|-----------------|  
|`S_OK`|`EnumTypeRefs`regresó con éxito.|  
|`S_FALSE`|No hay tokens para enumerar. En ese `pcTypeRefs` caso, es cero.|  
  
## <a name="remarks"></a>Observaciones  
 Un token TypeRef representa una referencia a un tipo.
