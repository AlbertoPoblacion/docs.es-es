# <a name="imetadataemit2savedeltatomemory-method"></a>IMetaDataEmit2::SaveDeltaToMemory (Método)
Guarda los cambios de la sesión de edición y continuación actual en la memoria.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT SaveDeltaToMemory (  
    [out] void        *pbData,
    [in]  ULONG       cbData  
);  
```  
  
## <a name="parameters"></a>Parámetros  
 `pbData`  
 [fuera] La dirección en la que comenzar a escribir el delta de metadatos.  
  
 `cbData`  
 [en] El tamaño de los cambios. Utilice [IMetaDataEmit2::GetDeltaSaveSize](../../../../docs/framework/unmanaged-api/metadata/imetadataemit2-getdeltasavesize-method.md) para determinar el tamaño.
