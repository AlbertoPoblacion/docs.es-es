# <a name="cleardownloadcache-function"></a>ClearDownloadCache (Función)
Borra la caché global de ensamblados de los ensamblados descargados.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT ClearDownloadCache ();  
```  
  
## <a name="requirements"></a>Requisitos  
 **Select** Consulte [Requisitos del sistema](../../get-started/system-requirements.md).  
  
 **Encabezado**: Fusion. h  
  
 **Biblioteca** Fusion. dll y mscorwks. dll. Use Fusion. dll en lugar de mscorwks. dll para asegurarse de que tiene como destino la versión correcta del .NET Framework.
