# <a name="iassemblycacheitemabortitem-method"></a>IAssemblyCacheItem::AbortItem (Método)
Permite que el ensamblado en la caché global de ensamblados realizar operaciones de limpieza antes de su lanzamiento.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT AbortItem ();  
```
