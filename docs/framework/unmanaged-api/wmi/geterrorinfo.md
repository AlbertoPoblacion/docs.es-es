# <a name="geterrorinfo-function"></a>Función GetErrorInfo
Recupera información sobre el error de la llamada de función anterior.  
  
[!INCLUDE[internalonly-unmanaged](../../../../includes/internalonly-unmanaged.md)]
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
IErrorInfo* GetErrorInfo();
```  

## <a name="return-value"></a>Valor devuelto

Un puntero a un [IErrorInfo](https://docs.microsoft.com/previous-versions/windows/desktop/api/oaidl/nn-oaidl-ierrorinfo) objeto si `null` la llamada de función se realiza correctamente o si se produce un error.
  
## <a name="remarks"></a>Observaciones

Esta función ajusta una llamada a la [IComThreadingInfo::GetErrorInfo](/windows/desktop/api/objidlbase/nf-objidlbase-icomthreadinginfo-getcurrentapartmenttype) método.
