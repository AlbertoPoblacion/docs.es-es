# <a name="connectionm_writelist-field"></a>Connection.m\_Campo WriteList

`Connection.m_WriteList`es <xref:System.Collections.ArrayList> una <xref:System.Net.HttpWebRequest> de las objetos que se ponen en cola para enviarse a través de HTTP.

## <a name="syntax"></a>Sintaxis
  
```csharp  
private ArrayList m_WriteList
```

> [!WARNING]
> El `Connection.m_WriteList` campo es privado y no está diseñado para usarse directamente en el código.
>
> Microsoft no admite el uso de este campo en una aplicación de producción bajo ninguna circunstancia.

## <a name="requirements"></a>Requisitos

**Espacio de nombres:**<xref:System.Net>

**Montaje:** Sistema (en System.dll)

**Versiones de .NET Framework:** Disponible desde 2.0.
