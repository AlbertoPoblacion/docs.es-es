# <a name="httpwebrequest_autoredirects-field"></a>HttpWebRequest. \_Campo de autoredirecciones

`HttpWebRequest._AutoRedirects`es <xref:System.Int32> un que refleja el número de <xref:System.Net.HttpWebRequest>redirecciones hechas para esto.

## <a name="syntax"></a>Sintaxis  
  
```csharp  
private int _AutoRedirects
```

> [!WARNING]
> El `HttpWebRequest._AutoRedirects` campo es interno y no está diseñado para usarse directamente en el código.
>
> Microsoft no admite el uso de este campo en una aplicación de producción bajo ninguna circunstancia.

## <a name="requirements"></a>Requisitos

**Espacio de nombres:**<xref:System.Net>

**Montaje:** Sistema (en System.dll)

**Versiones de .NET Framework:** Disponible desde 2.0.
