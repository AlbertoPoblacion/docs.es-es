# <a name="sqlstreamchars-constructor"></a>Constructor SqlStreamChars

Inicializa una nueva instancia de la clase `SqlStreamChars`. El ensamblado que contiene este constructor tiene una relación de confianza con SQLAccess. dll. Está pensado para su uso por SQL Server. En el caso de otras bases de datos, use el mecanismo de hospedaje proporcionado por esa base de datos.

```csharp
protected SqlStreamChars ();
```

## <a name="remarks"></a>Comentarios

> [!WARNING]
> El constructor `SqlStreamChars` está protegido y no está diseñado para usarse directamente en el código.
>
> Microsoft no admite el uso de este constructor en una aplicación de producción bajo ninguna circunstancia.
