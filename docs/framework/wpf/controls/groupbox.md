# <a name="groupbox"></a>GroupBox
El <xref:System.Windows.Controls.GroupBox> control es un <xref:System.Windows.Controls.HeaderedContentControl> que proporciona un contenedor titulado para el contenido de la interfaz gráfica de usuario (GUI).  
  
 En la ilustración siguiente se <xref:System.Windows.Controls.GroupBox> muestra un que <xref:System.Windows.Controls.TabControl> contiene un <xref:System.Windows.Controls.Button> y un que se incluyen <xref:System.Windows.Controls.StackPanel>en un.
