# <a name="separator"></a>Separador
Un <xref:System.Windows.Controls.Separator> control dibuja una línea, horizontal o vertical, entre los elementos en controles, como <xref:System.Windows.Controls.ListBox>, <xref:System.Windows.Controls.Menu>, y <xref:System.Windows.Controls.ToolBar>.
