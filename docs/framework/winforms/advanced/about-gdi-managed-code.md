# <a name="about-gdi-managed-code"></a>Código administrado de GDI+
GDI + es la parte del sistema operativo Windows que proporciona gráficos vectoriales bidimensionales, imágenes y tipografía. GDI + mejora GDI (la interfaz de dispositivo gráfico incluida en versiones anteriores de Windows) mediante la adición de nuevas características y optimizar las características existentes.  
  
 La interfaz de clases administradas de GDI + (un conjunto de contenedores) forma parte de .NET Framework, un entorno para compilar, implementar y ejecutar servicios Web XML y otras aplicaciones.  
  
 Esta sección proporciona información acerca de la API de GDI + para los programadores que utilizan código administrado.
