# <a name="radiobutton-control-windows-forms"></a>RadioButton (Control, formularios Windows Forms)
Windows Forms `RadioButton` controles presentan un conjunto de dos o más opciones mutuamente excluyentes para el usuario. Aunque los botones de radio y casillas de verificación puede parecer que funcionan de forma similar, hay una diferencia importante: cuando un usuario selecciona un botón de radio, los otros botones de radio en el mismo grupo no se puede seleccionar también.
