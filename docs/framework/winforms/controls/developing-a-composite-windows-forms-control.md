# <a name="develop-a-composite-windows-forms-control"></a>Desarrollar un control de Windows Forms compuesto

Puede desarrollar un control de Windows Forms compuesto mediante la combinación de otros controles de Windows Forms. Los controles compuestos que derivan de se denominan controles de <xref:System.Web.UI.UserControl> usuario. La clase base, <xref:System.Windows.Forms.UserControl>, proporciona el enrutamiento de teclado de los controles secundarios, lo que asegura que puedan recibir el foco. Para obtener un ejemplo de un control de usuario, <xref:System.Windows.Forms.UserControl> vea el [ejemplo de cómo: Aplicar atributos en controles](how-to-apply-attributes-in-windows-forms-controls.md)de Windows Forms.

El diseñador de Windows Forms de Visual Studio proporciona una compatibilidad enriquecida en tiempo de diseño para la creación de controles de usuario.
