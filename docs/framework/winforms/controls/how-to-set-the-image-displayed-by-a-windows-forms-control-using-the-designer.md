# <a name="how-to-set-the-image-displayed-by-a-windows-forms-control-using-the-designer"></a>Procedimiento Establecer la imagen mostrada por un control de Windows Forms mediante el diseñador

Varios controles Windows Forms pueden mostrar imágenes. La imagen puede ser un icono que clarifica el propósito del control, como un icono de disco en un botón que denota el comando **Guardar** . O bien, el icono puede ser una imagen de fondo para dar al control la apariencia deseada.

## <a name="to-set-the-image-displayed-by-a-control"></a>Para establecer la imagen mostrada por un control

1. En la ventana **propiedades** , seleccione la propiedad **Image** o **BackgroundImage** del control y, a continuación, seleccione los puntos![suspensivos (el botón de puntos](./media/visual-studio-ellipsis-button.png)suspensivos de Visual Studio) para mostrar el **recurso Select** . cuadro de diálogo.

2. Seleccione la imagen que desea mostrar.
