# <a name="how-to-split-a-window-horizontally-using-the-designer"></a>Procedimiento para dividir una ventana horizontalmente con el diseñador

Este ejemplo hace que el divisor que divide <xref:System.Windows.Forms.SplitContainer> el control horizontalmente.

> [!NOTE]
> La <xref:System.Windows.Forms.SplitContainer.Orientation%2A> propiedad<xref:System.Windows.Forms.SplitContainer> del control determina la dirección del divisor, no del propio control.

## <a name="to-split-a-window-horizontally"></a>Para dividir una ventana horizontalmente

En la ventana **propiedades** , establezca la <xref:System.Windows.Forms.SplitContainer.Orientation%2A> propiedad del <xref:System.Windows.Forms.SplitContainer> control en <xref:System.Windows.Forms.Orientation.Horizontal>.
