# <a name="errorprovider-component-windows-forms"></a>ErrorProvider (Componente, Formularios Windows Forms)
Los formularios de Windows `ErrorProvider` componente se utiliza para mostrar al usuario de forma no intrusiva que algo no funciona correctamente. Normalmente se utiliza junto con la validación de entrada del usuario en un formulario o mostrar errores dentro de un conjunto de datos.
