# <a name="folderbrowserdialog-component-windows-forms"></a>FolderBrowserDialog (Componente, Formularios Windows Forms)
Los formularios de Windows `FolderBrowserDialog` componente muestra una interfaz con la que los usuarios pueden examinar y seleccionar una carpeta o crear uno nuevo. Es un complemento para el [componente OpenFileDialog](openfiledialog-component-windows-forms.md) componente, que se utiliza para examinar y seleccionar los archivos.
