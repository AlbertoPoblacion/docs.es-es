# <a name="colordialog-component-windows-forms"></a>ColorDialog (Componente, formularios Windows Forms)
El componente <xref:System.Windows.Forms.ColorDialog> Formularios Windows Forms es un cuadro de diálogo preconfigurado que permite al usuario seleccionar un color de una paleta y agregar colores personalizados a esa paleta. Es el mismo cuadro de diálogo que se ve en otras aplicaciones basadas en Windows para seleccionar colores. Utilícelo en la aplicación basada en Windows como una solución sencilla, en lugar de configurar su propio cuadro de diálogo.
