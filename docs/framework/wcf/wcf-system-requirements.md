# <a name="wcf-system-requirements"></a>Requisitos del sistema WCF
Windows Communication Foundation (WCF) es una infraestructura de comunicación que se usa para crear aplicaciones distribuidas. A continuación enumeran los requisitos que permiten WCF ejecutar.  
  
## <a name="system-requirements"></a>Requisitos del sistema  
 WCF está instalado de forma predeterminada en [!INCLUDE[wv](../../../includes/wv-md.md)].  
  
 WCF también puede instalarse en [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], o [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].  
  
 **Tenga en cuenta** funcionalidad The Message Queuing (MSMQ) de WCF solo se admite en [!INCLUDE[wv](../../../includes/wv-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)], y [!INCLUDE[wxppro](../../../includes/wxppro-md.md)].
