# <a name="feedback-and-community"></a>Comentarios y comunidad
Agradecemos sus comentarios y preocupaciones sobre la documentación de Windows Communication Foundation (WCF) o CardSpace. En la parte inferior de cada página del SDK puede encontrar la frase "Envíe comentarios sobre este tema a Microsoft." Haga clic en el vínculo "comentarios" para enviar sus opiniones o sugerencias sobre el tema correspondiente.  
  
 Si se desea obtener más información, o desea compartir técnicas con otras personas interesadas en WCF, pruebe el [foro de MSDN Windows Communication Foundation](https://go.microsoft.com/fwlink/?LinkId=186463).
