Esta sección contiene ejemplos que muestran los transportes personalizados en Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>En esta sección  
 [Activación UDP](../../../../docs/framework/wcf/samples/udp-activation.md)  
 Muestra cómo extender el [transporte: UDP](../../../../docs/framework/wcf/samples/transport-udp.md) ejemplo para admitir la activación del proceso usando el servicio de activación de procesos de Windows (WAS).  
  
 [Transporte: Transacciones personalizadas sobre UDP](../../../../docs/framework/wcf/samples/transport-custom-transactions-over-udp-sample.md)
