# <a name="queued-poison-messages"></a>Mensajes dudosos en cola
Nombre del contador: En la cola de mensajes dudosos.  
  
## <a name="description"></a>Descripción  
 El número de mensajes que son marcados como dudosos por el transporte en cola en este servicio.  
  
 Consulte [de mensajes dudosos](https://go.microsoft.com/fwlink/?LinkID=96233) para obtener más información sobre cuándo los mensajes se marcan como dudosos.
