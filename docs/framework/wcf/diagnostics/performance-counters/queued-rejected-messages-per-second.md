# <a name="queued-rejected-messages-per-second"></a>Mensajes en cola rechazados por segundo
Nombre del contador: Mensajes en cola rechazados por segundo.  
  
## <a name="description"></a>Descripción  
 Número de mensajes rechazados cada segundo por el transporte en cola en este servicio.  
  
 Este contador es de tipo de contador de rendimiento [PERF_COUNTER_COUNTER](https://go.microsoft.com/fwlink/?LinkID=94649), cuyo valor se calcula mediante la siguiente fórmula.  
  
 (N 1 - N 0 ) / ( (D 1 -D 0 ) / F)
