# <a name="queue-dropped-messages-per-second"></a>Mensajes en cola quitados por segundo
Nombre de contador: Mensajes en cola quitados por segundo.  
  
## <a name="description"></a>Descripción  
 Número de mensajes quitados cada segundo por el transporte en cola en este servicio.  
  
 Este contador es de tipo de contador de rendimiento [PERF_COUNTER_COUNTER](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2003/cc740048(v=ws.10)), cuyo valor se calcula mediante la siguiente fórmula.  
  
 (N 1 - N 0 ) / ( (D 1 -D 0 ) / F)
