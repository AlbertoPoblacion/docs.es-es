# <a name="systemservicemodelchannelshttpsclientcertificateinvalid"></a>System.ServiceModel.Channels.HttpsClientCertificateInvalid
Certificado de cliente no válido.  
  
## <a name="description"></a>Descripción  
 Este seguimiento de traza indica que el agente de escucha HTTPS consideró no válido el certificado proporcionado por el cliente. El agente de escucha HTTPS intentaba validar la autenticidad del cliente utilizando este certificado. Un certificado puede no ser válido si la entidad de certificación no está reconocida por el servidor que hospeda el servicio.
