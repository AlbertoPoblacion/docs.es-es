# <a name="debugging-on-the-client"></a>Depuración del cliente
Para facilitar a los usuarios escribir aplicaciones de cliente para el servicio WCF, puede agregar el [ \<serviceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) comportamiento al archivo de configuración del servicio del servicio. Este comportamiento se puede utilizar para publicar páginas de ayuda y devolver información de excepciones administrada en los detalles de errores SOAP devueltos al cliente.
