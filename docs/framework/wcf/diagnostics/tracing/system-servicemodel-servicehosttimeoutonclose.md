# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose
System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Descripción  
 La operación de cierre de ServiceHost agotó el tiempo de espera. Esto indica que algunos canales con sesión no fueron correctamente apagados por el cliente y se podrían haber anulado. Modifique el tiempo predeterminado para ajustar el tiempo de espera de cierre.
