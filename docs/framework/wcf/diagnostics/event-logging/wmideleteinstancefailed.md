# <a name="wmideleteinstancefailed"></a>WmiDeleteInstanceFailed
Id.: 122  
  
 Gravedad Error  
  
 Categoría ServiceModel  
  
## <a name="description"></a>DESCRIPCIÓN  
 Este evento indica que no se procesó la solicitud DeleteInstance de WMI. El evento enumera la excepción, el nombre del proceso y el id. del mismo.
