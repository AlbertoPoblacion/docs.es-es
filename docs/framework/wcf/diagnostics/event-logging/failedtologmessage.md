# <a name="failedtologmessage"></a>FailedToLogMessage
Id.: 113  
  
 Gravedad: Error  
  
 Categoría: ServiceModel  
  
## <a name="description"></a>Descripción  
 Este evento indica que un mensaje no se registró. El evento enumera la excepción, el nombre del proceso y el id. del mismo.
