# <a name="missingnecessarykeyusage"></a>MissingNecessaryKeyUsage
Id.: 156  
  
 Gravedad: Error  
  
 Categoría: TransactionBridge  
  
## <a name="description"></a>Descripción  
 Este evento indica que El certificado de identidad con el nombre de sujeto especificado y la huella digital no proporciona datos concretos entre KeyUsages. También enumera el nombre y el id. del proceso.
