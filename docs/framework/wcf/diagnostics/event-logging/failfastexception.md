# <a name="failfastexception"></a>FailFastException
Id.: 103  
  
 Gravedad: Error  
  
 Categoría: Traza  
  
## <a name="description"></a>Descripción  
 Se inició una excepción durante FailFast. El evento enumera la excepción, el nombre del proceso y el id. del mismo.
