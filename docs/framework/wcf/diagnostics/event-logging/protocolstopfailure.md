# <a name="protocolstopfailure"></a>ProtocolStopFailure
Id.: 147  
  
 Gravedad: Error  
  
 Categoría: TransactionBridge  
  
## <a name="description"></a>Descripción  
 Este evento indica que el servicio de protocolo WS-Atomic Transaction no se pudo detenerse. El evento indica el identificador de protocolo, el nombre de protocolo, la excepción, el nombre del proceso y el identificador del proceso.
