# <a name="failedtoloadperformancecounter"></a>FailedToLoadPerformanceCounter
Id.: 118  
  
 Gravedad Error  
  
 Categoría ServiceModel  
  
## <a name="description"></a>DESCRIPCIÓN  
 Este evento indica que no se cargó un contador de rendimiento. El evento enumera el nombre de la categoría, el nombre del contador, el nombre de la excepción, el nombre del proceso y el id. de proceso.
