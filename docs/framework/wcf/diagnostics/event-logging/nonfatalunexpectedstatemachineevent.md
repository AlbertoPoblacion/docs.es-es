# <a name="nonfatalunexpectedstatemachineevent"></a>NonFatalUnexpectedStateMachineEvent
Id.: 148  
  
 Gravedad: Error  
  
 Categoría: TransactionBridge  
  
## <a name="description"></a>Descripción  
 Este evento indica que un equipo de estado intentó procesar un evento inesperado. El evento no se considera grave. El evento hace una lista del id. de la transacción, equipo de estados, el estado actual, historial, el nombre de evento, datos de evento, nombre de proceso e id. de proceso.
