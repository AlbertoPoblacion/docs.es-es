# <a name="thumbprintnotvalidated"></a>ThumbPrintNotValidated
Id.: 153  
  
 Gravedad Error  
  
 Categoría TransactionBridge  
  
## <a name="description"></a>DESCRIPCIÓN  
 Este evento indica que no se pudo validar un certificado de identidad con la huella digital especificada. El evento enumera el nombre y el id. de proceso.
