# <a name="entity-framework-data-providers"></a>Proveedores de datos de Entity Framework
En esta sección se proporciona información sobre los proveedores de datos que admiten [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)].  
  
## <a name="in-this-section"></a>En esta sección  
 [Proveedor de EntityClient para Entity Framework](../../../../../docs/framework/data/adonet/ef/entityclient-provider-for-the-entity-framework.md)  
 Describe el proveedor de datos de EntityClient. Este proveedor transforma en un árbol de comandos canónico las consultas realizadas en un modelo de datos. El árbol de comandos, a continuación, puede utilizarse en un proveedor de datos de .NET Framework para la [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)].  
  
 [SqlClient para Entity Framework](../../../../../docs/framework/data/adonet/ef/sqlclient-for-the-entity-framework.md)  
 Describe el proveedor de datos de .NET Framework que admite el [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] para su uso con una base de datos de SQL Server.
