> [!NOTE]
> Los ejemplos de C# de este artículo se ejecutan en el ejecutor de código en línea y área de juegos de [Try.NET](https://dotnet.microsoft.com/platform/try-dotnet). Haga clic en el botón **Ejecutar** para ejecutar un ejemplo en una ventana interactiva. Una vez que se ejecuta el código, puede modificar y ejecutar el código modificado si vuelve a hacer clic en **Ejecutar**. El código modificado se ejecuta en la ventana interactiva o, si se produce un error en la compilación, en la ventana interactiva se muestran todos los mensajes de error del compilador de C#.
