Para usar el carácter de subrayado como separador inicial, debe agregar el elemento siguiente al archivo del proyecto de Visual Basic (\*.vbproj):

```xml
<PropertyGroup>
  <LangVersion>15.5</LangVersion>
</PropertyGroup>
```

Para obtener más información, consulte [configuración de la versión](../docs/visual-basic/language-reference/configure-language-version.md)del lenguaje Visual Basic .
